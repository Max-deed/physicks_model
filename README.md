# physicks_model
Ввести понятие полезной работы, затраченной работы и коэффициента полезного действия (КПД) на конкретном примере

Задача: Разработайте модель колебаний тела в желобе. Модель должна учитывать силу трения.

Функционал модели: 1. Визуализация: тело (точка=шарик), параболический желоб. Показывается вектор силы тяжести. 2. Управление: Задание массы тела, начальной высоты подъема, задание коэффициента трения. 3. Графически отобразите, как затраченная работа «распадается» на три части: полезную работу, работу против трения и (если есть) работу по сообщению кинетической энергии. 4. Постройте анимацию колебаний точки в таком желобе (Рис. ниже).

От Ромы:
Инициализация: Model(m, h0, mu)

шаг симуляции: model.step(dt)

Текущее состояние: state = model.get_state()

модуль должен возвращать 
state = {
    
    "t": float,            # время
    "x": float,            # координата x
    "y": float,            # координата y

    "v": float,            # скорость (опционально, но лучше есть)

    "Ek": float,           # кинетическая энергия
    "Ep": float,           # потенциальная энергия

    "A_friction": float,   # работа трения (накопленная)
    "A_kinetic": float,    # работа на кинетическую энергию
    "A_useful": float      # полезная работа
}

От Дениса:

Надо использовать state = model.get_state()
