# physicks_model
Ввести понятие полезной работы, затраченной работы и коэффициента полезного действия (КПД) на конкретном примере

Задача: Разработайте модель колебаний тела в желобе. Модель должна учитывать силу трения.

Функционал модели: 1. Визуализация: тело (точка=шарик), параболический желоб. Показывается вектор силы тяжести. 2. Управление: Задание массы тела, начальной высоты подъема, задание коэффициента трения. 3. Графически отобразите, как затраченная работа «распадается» на три части: полезную работу, работу против трения и (если есть) работу по сообщению кинетической энергии. 4.
